Requirements:
MariaDB or MySQL installed on the computer

Python modules:
> pyglet
> csv
> mysql.connector
> tkinter

System:
> Screen Resolution - 1920x1080 recommended


Set up

1. Edit lines 4,5,6 in DataEntry.py to match the username, password and working database.
2. Change port number from the default(3306) on line 9 in DataEntry.py if necessary or else skip this step.

3. Edit lines 9,10,11 in ElementBackBone.py to match the username, password and working database.
4. Change port number from the default(3306) on line 14 in ElementBackbone.py if necessary or else skip this step.

5. Run DataEntry.py after going through all the necessary steps.

6. Make sure that the following files are all in the same working directory:
	> DataEntry.py
	> ElementBackBone.py
	> main.py
	> PeriodicTable.csv
	> MathInfo.csv
	> TypeInfo.csv
	> ElementExtra.csv
 

You're ready to go! Run main.py for all that elemental glory :)
